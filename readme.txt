== Description ==

**Google Analytics Tester** is a lightweight yet powerful WordPress plugin that lets you quickly check if Google Analytics tracking code is installed on any website.  

Built with a **modern frontend design** and **AJAX-powered results**, it provides a smooth, user-friendly experience without requiring a page reload.  

### Key Features
* Detects **Google Analytics 4 (GA4)** and **Universal Analytics (UA)**.
* AJAX functionality → instant results without page refresh.
* Clean, responsive UI with full-width form styling.
* Easy shortcode: `[ga_tester_form]` can be added anywhere.
* Clear success and error messages.
* Built-in styles that work independently of your WordPress theme.

This tool is perfect for **SEO specialists, web developers, digital marketers, and business owners** who want to verify if tracking codes are working properly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/google-analytics-tester` directory, or install it directly via the WordPress plugin dashboard.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Add the shortcode `[ga_tester_form]` to any page or post.
4. Visit the page, enter a website URL, and instantly check if Google Analytics is detected.

== Frequently Asked Questions ==

= How do I display the form? =
Simply use the shortcode `[ga_tester_form]` inside a page or post.

= What does it detect? =
It detects both **Google Analytics 4 (GA4)** and **Universal Analytics (UA)** tracking codes.

= Will it interfere with my site’s analytics? =
No. The plugin only scans the HTML of the entered URL. It does not alter or affect your own analytics tracking.

= Can I customize the style? =
The plugin comes with a clean, full-width design that works out of the box. For further customization, you can override the CSS.
